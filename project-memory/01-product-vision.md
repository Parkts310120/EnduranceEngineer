# Endurance Engineer

## Missão

Construir o sistema operacional definitivo para equipes de endurance racing.

O Endurance Engineer deve centralizar toda a operação de uma equipe em um único software.

Nossa meta é eliminar a necessidade de ferramentas espalhadas como Excel, Google Sheets, Google Docs, Notion e Discord para a gestão da equipe.

---

# Visão

O produto será um SaaS multi-equipes capaz de operar desde pequenas equipes até organizações profissionais de endurance.

Cada Workspace representa uma operação completa de corrida.

Dentro dele estarão todas as informações necessárias antes, durante e após a corrida.
